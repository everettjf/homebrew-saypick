# homebrew-saypick

Homebrew tap for [**Saypick**](https://github.com/everettjf/Saypick) — system-wide AI translation & inline rewrite for macOS.

## Install

```bash
brew install --cask everettjf/saypick/saypick
```

Or tap first, then install:

```bash
brew tap everettjf/saypick
brew install --cask saypick
```

## Update

```bash
brew upgrade --cask saypick
```

## Uninstall

```bash
brew uninstall --cask saypick
```

## Requirements

- macOS 15 (Sequoia) or later
- Saypick is signed and notarized by Apple.

After installing, grant **Accessibility** permission in System Settings → Privacy & Security → Accessibility so Saypick can read selections and replace text.
